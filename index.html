CSS BASICS

HTML stands for Hypertext Markup Language. It is the standard language used to create and structure web pages on the internet. HTML uses tags to define elements such as headings, paragraphs, links, and images, allowing browsers to display content correctly. It forms the backbone of most websites, enabling easy information presentation and navigation.
